How It Works

From zero to orchestrated in minutes

One Docker command. A guided setup wizard. Your AI team is operational.

1

Install

One command. Docker Compose pulls everything: app, bridge, database. Done in under a minute.

2

Configure

The setup wizard walks you through: company profile, AI providers, departments, and agent roles.

3

Deploy

Agents are live. Heartbeats running. Routines scheduled. Your AI team starts working autonomously.

4

Orchestrate

Monitor, direct, and scale from the dashboard. Set goals, review work, manage budgets. Stay in control.

Capabilities

Everything you need to run an AI-powered organization

Not a chatbot. Not a dev tool. An operating system for AI teams.

Departments

Mirror your real org structure. Engineering, Design, Research, Operations -- each department gets its own agents, tasks, and budget.

Heartbeat Engine

Agents work autonomously on a schedule. Morning briefs, daily research, code reviews -- they execute without being asked.

Multi-Model Router

Claude CLI, OpenRouter, OpenAI, Perplexity. Route each agent to the best model for the job. CLI-first economics keep costs predictable.

Guided Configuration

Step-by-step setup wizard. Department templates with pre-configured agents. Smart defaults. No YAML files, no guesswork.

Routines

Multi-step agent workflows on cron, webhook, or manual triggers. Chain agents together. Track every run, every token, every output.

Cost Control

Per-agent, per-department, and company-wide budgets. Automatic warnings and hard stops. CLI savings tracking. Full cost transparency.

Who Is This For

Team Leads
